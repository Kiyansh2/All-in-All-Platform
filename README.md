## GitHub Pages deployment

1. Upload all files and folders to your repository root.
2. Go to `Settings > Pages`.
3. Select `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/(root)`.
6. Save and wait for the deployment to turn green.
